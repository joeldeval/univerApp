# univerApp
App realizada con 

  * [Angular][1]
  * [PHP][2] API
  * [MariaDB][3]

Instalar un servidor apache:
    
    sudo yum install httpd.

Iniciar el servidor:

    sudo service httpd start.


[1]: https://angular.io/    "AngularJS"
[2]: http://php.net/        "PHP"
[3]: https://mariadb.org/   "MariaDB"
